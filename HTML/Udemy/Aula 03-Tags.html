A tag "bold" serve unicamente pra deixar um texto em negrito. Já a tag "strong" faz o mesmo, mas com um sentido mais lógico
Enquanto isso, temos um caso semelhante utilizando a tag "i", para deixar algo fisicamente em itálico, mas a tag "em", tem a mesma funcionalidade mas é utilizado para dar ênfase.

Link

Semelhante aos casos de negritos e itálicos temos as tags a seguir: A tag "del" vai indicar um erro com valor semântico, enquando a tag "s" fará o mesmo com a palavra mas dessa vez tem um valor físico. Outros casos temos com as tags "ins" e "u", as quais fazem a mesma coisa, porém a tag "ins" tem valor sintático enquanto a tag "u" tem valor físico.

Outras tags em HTML

Em texto pequeno utiliza-se a tag "small", para deixar algo elevado utilizamos "sup", para deixar uma palavra na parte inferior utiliza-se "sub", pra fazermos uma citação usamos "q", e por ultimo temos a tag "code" para indicar um código

Últimas tags:

Por últimos teremos as tags "span" e "div", as duas servem como conteiners genericos, a diferença é que a tag de span é inline, podendo ser utilizadas dentro de parágrafos diferente da tag div que é um bloco.
